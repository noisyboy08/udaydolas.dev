## Overview

Netlytics is an AI-powered Netflix analytics and visualization platform that provides deep insights into content trends, viewer preferences, and strategic analytics for movies and TV shows from 2008-2021.

## Key Features

- **📊 Advanced Content Analytics Dashboard** - Comprehensive data visualization
- **🎬 Movie & TV Show Insights** - Detailed analysis of content (2008-2021)
- **📈 KPI Cards and Growth Trends** - Visual representation of key metrics
- **🎨 Beautiful Data Visualizations** - Interactive charts and graphs
- **🤖 AI-Assisted Strategic Insights** - Machine learning-powered recommendations

## Tech Stack

- **Frontend**: React, TypeScript
- **Backend**: Supabase
- **Analytics**: Jupyter Notebooks
- **Data Processing**: Python automation scripts
