## Overview

AIRBNB-360 is an AI intelligence platform for Airbnb data that transforms CSV files into a comprehensive predictive analytics dashboard. The platform uses machine learning to generate multiple scores per listing and provides deep market insights through a stunning, cinematic interface.

![AIRBNB-360 dashboard showing predictive Airbnb insights](/images/blog/airbnb360.png)

## Key Features

- **🎯 8+ ML Scores Per Listing** - Trust/Risk Analysis & NLP Host Personality classification
- **🔮 Predictive Analytics Engine** - Advanced forecasting capabilities
- **⭐ Rising Stars Identification** - Host Growth Predictor to identify emerging hosts
- **🎨 Cinematic UI/UX Design** - Spectral gradients, aurora animations, and 3D glassmorphism
- **📊 Deep Market Insights** - Comprehensive data visualization and analysis

## Tech Stack

- **Frontend**: TypeScript, React
- **AI/ML**: Machine Learning models
- **NLP**: Natural Language Processing for personality analysis
- **Data Visualization**: Advanced charting and visualization libraries
